What Is Image Generation in 4o

As you could guess, 4o supports native image generation. This means you can describe an image in text and receive a visual output directly inside the ChatGPT interface without needing a separate tool or model.
Also, this feature allows for fast, precise, and practical image creation - from infographics and diagrams to photorealistic scenes and comic strips.

Where to Find It

To use image generation, you should ensure that you're using 4o model. Next, click on the "+" icon next to the message bar, select Create image and type your prompt into the input box. Then, the model will return an image within seconds.

Differences from DALL-E

You probably heard about DALL-E, a model that previously was the OpenAI's standalone image generator.
However, with the Create Image mode being integrated, it no longer is. In comparison to DALL-E and, in fact, any other model, this mode integrates image generation directly into its multimodal interface.
In other words, it makes the process much faster and more interactive. Also, what makes 4o image generation different is the fact that it maintains visual consistency with the textual request better than any other model, including DALL-E.
Hence, it even allows building complex scenes using its built-in knowledge.
This feature excels at creating useful imagery - logos, signage, instructions, posters, and more - not just artistic or surreal illustrations. For example, it can render accurate street signs or readable menus with detailed text.
