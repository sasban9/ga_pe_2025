05. Basic Prompt Structure and Key Parts

Prompts are made up of several key components that work together to guide the AI. While not every prompt will contain all these elements, understanding how each part functions can help you create more targeted and effective inputs.

The Key Parts of a Prompt

  • The Directive
  • Examples
  • Role / Persona
  • Output Formatting
  • Additional Information

1. The Directive

The Directive is the main instruction in the prompt. It tells the AI exactly what task it should perform.

What is a Directive?

A directive is a concise instruction or question that gives the AI a clear task to perform.

Tell me five good books to read.

Implicit Directives

Some prompts imply a directive:

Night: Noche Morning:

Best Practices for Directives

  • Be clear and concise.
  • Avoid ambiguous instructions.
  • Use action verbs: “write,” “list,” “translate.”

2. Examples

When the task is more complex, providing Examples can help guide the AI in producing more accurate responses.

Translate the following sentences:
Q: I like apples. A: Me gustan las manzanas.
Q: I enjoy walking.

Best Practices for Examples

  • Provide clear and relevant examples.
  • Use examples to show expected structure.
  • Use one-shot or few-shot based on complexity.

3. Role (Persona)

Assigning a Role to the AI frames the response in a specific way. This can guide tone, style, and content.

You are a doctor. Diagnose the patient based on these symptoms: Fever, cough, and fatigue.
You are a customer service agent. Write an email apologizing for a delayed order.

Best Practices for Roles

  • Use roles to add expertise or tone.
  • Ensure the role fits the task.
  • Combine with examples/context when needed.

4. Output Formatting

Output Formatting ensures that the response follows a specific structure, like a table, CSV, or paragraph.

Output this information as a CSV:
Case: 2024_ABC...
Client: XYZ Corp...
Case,Client,Jurisdiction,Filed Date,...

Style Instructions

You can also specify tone, formality, or length:

Write a clear and curt paragraph about llamas.

5. Additional Information

Additional information gives background the AI needs to complete the task accurately.

January 1, 2000: Fractured right arm...
You are a doctor. Predict future risks.

Best Practices for Additional Info

  • Include only what's relevant.
  • Link information clearly to the task.
  • Use when context is essential.

Recommended Prompt Order

  1. Examples (if needed)
  2. Additional Information
  3. Role
  4. Directive
  5. Output Formatting

Conclusion

By mastering Directives, Examples, Roles, Formatting, and Additional Information, you can create effective prompts that guide AI to generate accurate and helpful responses.
